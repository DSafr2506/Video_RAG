# Changelog

## [0.1.0] - 2024-03-14

### Добавлено
- Базовая реализация системы Video RAG
- Модуль для разбиения транскриптов на чанки
- Интеграция с моделью BGE-M3 для эмбеддингов
- FAISS индексы для быстрого поиска
- Поддержка группировки по топикам
- Расширение временных окон
- Батчевая обработка запросов
- Документация и примеры использования

### Особенности
- Поддержка GPU
- Настраиваемые параметры
- Модульная архитектура
- Простота использования
