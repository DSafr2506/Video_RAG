# Формат данных

Эта директория должна содержать следующие файлы:

## transcripts.pkl

Pickle файл со словарем транскриптов:

```python
{
    "video1.mp4": [
        {
            "text": "Текст сегмента",
            "start": 0.0,  # время начала в секундах
            "end": 5.2     # время окончания в секундах
        },
        ...
    ],
    "video2.mp4": [...],
    ...
}
```

## video_topic.csv

CSV файл с соответствием видео и тематик:

```csv
video_file,topic
video1.mp4,технологии
video2.mp4,спорт
video3.mp4,технологии
```

## test_topic.csv

CSV файл с тестовыми запросами:

```csv
query_id,question,topic
1,Как работает искусственный интеллект?,технологии
2,Правила игры в футбол,спорт
```

## Пример создания данных

```python
import pickle

# Создание транскриптов
transcripts = {
    "video1.mp4": [
        {"text": "Привет всем", "start": 0.0, "end": 2.5},
        {"text": "Сегодня поговорим о Python", "start": 2.5, "end": 5.0}
    ]
}

with open("data/transcripts.pkl", "wb") as f:
    pickle.dump(transcripts, f)
```

## Примечания

- Все временные метки в секундах
- Текст должен быть в UTF-8
- Видео файлы должны иметь уникальные имена
